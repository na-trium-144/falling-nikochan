import { Key } from "@/common/key.js";
import { Move } from "@icon-park/react";

- 現在のカーソル位置に音符を追加し、音符の位置と飛んでくる角度・速度を変更できます。
- 音符の位置や速度は数値で指定する代わりに、
  左に表示されている音符のプレビューからマウス操作で変更することもできます。
  - 現在のカーソル位置に音符がある状態でプレビュー画面内をドラッグすると音符の位置が動き、<Key className="text-sm p-0.5 mx-0.5">Shift</Key>を押しながらドラッグすると音符の速度が変わります。
  - タッチ操作の場合は <span className="inline-block w-5 translate-y-0.5 "><Move /></span>タッチ操作モード
  ボタンを押すと音符の位置/速度をタッチ操作で動かせるモードに切り替えることができます。
- 下にある コピー、貼り付け、1 〜 9 のボタンは音符のコピー、貼り付けをすることができます。
  - 下段のボタンで現在のカーソル位置にある音符の設定をコピーします。
  - それぞれ対応する上段のボタンを押すと現在のカーソル位置にある音符に上書きして貼り付けます。
  - または音符がないときに貼り付けした場合は音符が追加されます。
