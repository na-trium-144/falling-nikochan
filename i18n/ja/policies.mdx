# 利用者の情報について

- Falling Nikochan はユーザー登録なしで利用することができます。利用者に個人情報の入力を求めることはありません。
- Falling Nikochan で遊んだプレイ記録は、匿名でサーバーに送信され、保存されます。これには Cookie や IP アドレス等の個人情報は含まれません。
- 利用者がプレイした譜面の履歴 (トップページの「プレイする」ページに一覧表示されるもの) や、ベストスコアなどのプレイ記録は、利用者のブラウザーの Local Storage に保存されます。これらの情報がサーバーに送信されることはありません。
- 利用者が譜面を作成しアップロードした場合には、利用者の IP アドレスが収集、保存されます。一部のAPIへのレート制限に用いられるほか、開発者が必要と判断した場合にはその他の目的に使用されることがありますが、これが第三者に提供されることはありません。

# YouTube

- 本サービスは [YouTube IFrame Player API](https://developers.google.com/youtube/iframe_api_reference) を利用しています。
  - YouTube IFrame Player API は YouTube が公式に提供している動画埋め込み機能であり、YouTube の利用規約に準拠した形で YouTube 上の動画を直接再生しています。
  非公式のツールなどによる動画のダウンロードや保存とは異なります。
- 本サービスに埋め込まれた動画の権利は元の動画の投稿者 (またはその動画で使用しているコンテンツの著作権者) にあります。
  動画の再生等により本サービスが収益を得ることはありません。
- Fallling Nikochan は YouTube とは無関係なサービスです。本サービスについて YouTube に問い合わせないでください。

# 譜面のアップロードについて

- 本サービスではユーザー登録なしで誰でも譜面をアップロードすることができます。
  利用者がアップロードした譜面は、 Falling Nikochan のサーバーに保存されます。
- 譜面作成時には、編集用パスワードを設定することができます。
- パスワードは編集に使用したブラウザーに保存させることができます。
  その場合通常はパスワードを入力しなくても編集できますが、
  PC やブラウザを変更した場合、またブラウザーに保存されたデータを削除した場合などには、パスワードの入力が求められることがあります。
- パスワードの保存には Local Storage および Cookie を使っています。個人情報を特定する用途では用いられません。
- Falling Nikochan ではパスワードを忘れた場合再設定する方法は用意しておりません。
  利用者の責任で管理してください。
- 本サービスは個人で運営しているサービスであるため、サーバーに保存されているデータについて機密性、完全性、可用性は保証できません。
  悪意のある第三者によって、または開発者の過失によってデータが流出したり消失したりした場合、利用者が被った不利益や損害について、開発者は責任を負いません。
  データのバックアップなどは利用者の責任で行ってください。
